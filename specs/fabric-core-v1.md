# Fabric Core Specification v1.0

## Canonical Records

- Entity
- Relationship
- Assertion
- Observation
- Projection
- Drift
- Reconciliation

## Mandatory Verbs

- ASSERT
- OBSERVE
- VERIFY
- PROJECT
- COMPARE
- RECONCILE

## Kernel Boundary

Identity Service
Assertion Service
Verification Service
Projection Service
Drift Service
Reconciliation Service

Everything else is an extension.
